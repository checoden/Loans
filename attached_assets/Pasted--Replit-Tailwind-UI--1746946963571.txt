Создай мобильное веб-приложение «Займы онайн» на платформе Replit, используя Tailwind UI. Приложение должно иметь:

1. Главную страницу со списком предложений от микрофинансовых организаций (МФО). Каждый элемент списка содержит:
   - Логотип МФО
   - Название компании
   - Максимальная сумма займа (хранится в БД только числом, например: "30000", отображается как "до 30000 ₽")
   - Срок займа от и до (хранится отдельно, например: от 5 до 30 дней)
   - Процентная ставка (хранится в БД только числом, например: "0". Если "0", то отображается яркий лейбл "0% первый займ")
   - Кнопку «Получить займ», при нажатии которой открывается партнерская ссылка МФО во встроенном WebView (без перехода в отдельный браузер).

Пример формата JSON из API:
[
  {
    "id": 1,
    "name": "Займер",
    "logo": "https://example.com/zaymer.png",
    "amount": 30000,
    "term_from": 5,
    "term_to": 30,
    "rate": 0,
    "is_first_loan_zero": true,
    "link": "https://partnerlink.com/zaymer",
    "priority": 1
  }
]

2. Компактные и удобные фильтры по сумме и сроку займа вверху страницы:
   - Поля ввода: «Сумма», «Срок займа» (ползунки).
   - Автоматическое обновление списка после ввода параметров.

3. API для получения данных предложений:
   - Реализуй простой бэкенд на Node.js с использованием Express (на Replit), который возвращает JSON с предложениями МФО.

4. Калькулятор переплаты:
   - Создай отдельную страницу или всплывающее окно с простым калькулятором, позволяющим ввести сумму и срок займа, после чего показывать ориентировочную сумму возврата и переплату.

5. Административную панель (админку):
   - Защищённую админку, доступную через отдельную авторизацию.
   - Возможность добавлять, редактировать и удалять предложения МФО.
   - Загрузка изображений (логотипов МФО).
   - Установка приоритета отображения предложений (чем выше приоритет, тем выше предложение в списке).

6. Поддержка Push-уведомлений:
   - Интегрируй сервис OneSignal для отправки push-уведомлений.

7. Современное меню (Header):
   - Размести меню в верхней части экрана (Header).

8. Подробная логика поведения пользователя:
   - Пользователь открывает приложение и сразу видит предложения микрозаймов.
   - Пользователь может воспользоваться фильтрами, указав сумму и срок займа, после чего список предложений автоматически обновляется.
   - Если предложение подразумевает ставку 0%, отображается специальный яркий лейбл «0% первый займ».
   - Пользователь нажимает на кнопку «Получить займ» и переходит к оформлению займа по партнерской ссылке внутри встроенного WebView.
   - Пользователь может воспользоваться калькулятором займа, чтобы предварительно рассчитать переплату.

9. Адаптивный интерфейс под мобильные устройства с использованием компонентов и стилистики Tailwind UI.

10. Подготовь проект таким образом, чтобы его можно было легко экспортировать и упаковать в APK-файл через AppsGeyser или аналогичные сервисы для последующей публикации в RuStore.
